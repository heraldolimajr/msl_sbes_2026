# Systematic Mapping Study – Raw Data and Artifacts (SBES 2026)

This repository provides the complete set of raw data and supporting artifacts associated with the Systematic Mapping Study (SMS) submitted to SBES 2026. Its purpose is to ensure transparency, reproducibility, and openness throughout the research process.

## Overview

The materials available here document all stages of the study, from the initial screening of studies to the final data extraction. All artifacts are provided as PDF files and are available directly in the root directory of this repository to facilitate access and anonymous review.

## Repository Structure

All files are organized in the root directory and correspond to different stages of the study:


Initial Screening (Titles and Abstracts)
PDF files containing the spreadsheets used during the first screening phase, including inclusion/exclusion decisions and duplicate removal.


Full-Text Screening (Detailed Analysis)
PDF files with the results of the full-text reading phase, including refined selection decisions and justification notes.


Data Extraction
PDF files corresponding to the data extraction forms and the extracted dataset from the selected primary studies.


Research Protocol (if applicable)
PDF document describing the study design, research questions, search strings, and methodological procedures.


Additional Documentation (if applicable)
Supplementary materials that support the understanding and reuse of the dataset.


Methodological Notes
This dataset was produced following established guidelines for Systematic Mapping Studies in Software Engineering. The process includes:


Definition of research questions


Construction and execution of search strings across multiple digital libraries


Study selection through a multi-stage screening process


Data extraction based on predefined criteria


All decisions made during screening and extraction are documented in the corresponding PDF files.
Reproducibility
The repository is structured to support reproducibility. Researchers can:


Trace each included study back to its screening decision


Verify how inclusion and exclusion criteria were applied


Reanalyze the extracted data


Reuse the dataset in secondary studies or meta-analyses


Anonymization
This repository is provided in an anonymized format to comply with double-blind review requirements. Any identifying information about the authors or institutions has been removed or generalized.
